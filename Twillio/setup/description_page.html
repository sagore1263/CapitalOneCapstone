University of Wisconsin–Madison
Capstone Fraud Detection System


This project is a University of Wisconsin–Madison capstone initiative focused on fraud detection and alerting. The system analyzes simulated financial activity and sends SMS notifications to users when suspicious activity is detected. Users are able to set a threshold between 0.0 (least likelihood of fraud) and 1.0 (highest likelihood of fraud) for their fraud alerting.

Users may voluntarily opt in to receive fraud alert text messages. Message frequency varies based on detected activity and user-selected alert thresholds. Message and data rates may apply. Users may reply STOP at any time to opt out, or HELP for assistance.

This system is developed for academic purposes as part of a student capstone project and does not process real financial transactions.
